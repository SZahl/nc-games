# Welcome to my NC Games Front End React project

To run the project locally:

    - Clone this GitHub repo.
    - Install all dependencies, by running 'npm install'.
    - Run 'npm start' to open in your browser.

Project Overview

    - This project was created to showcase my recently learnt front-end development skills. Created in React and demonstrating knowledge in React Hooks, Axios, CSS styling, Error Handling and more.

    - NC Games was created with mobile first design in mind, designing a comfortable user experience in both app and browser format.

    - Backend Server can be found here: https://github.com/SZahl/be-nc-games
